# Adult 데이터 코드북

## 1. 데이터 개요

- 데이터명: Adult(Census Income) 데이터
- 제공 기관: UCI Machine Learning Repository
- 원자료: 1994년 미국 인구조사 자료
- 분석 목적: 개인의 연간 소득이 50,000달러를 초과하는지 분류
- 현재 사용 파일: `adult.data`
- 원본 데이터 크기: 32,561행, 15열
- 결측 행만 제거한 크기: 30,162행, 15열
- 결측 행과 중복 행을 모두 제거한 최종 크기: 30,139행, 15열
- 입력 변수: 14개
- `income`: 연간 소득이 50,000달러 이하인지(`<=50K`), 초과인지(`>50K`)를 나타내는 목표 변수

공식 자료: [UCI Adult 데이터](https://archive.ics.uci.edu/dataset/2/adult)

변수 원문: [adult.names](https://archive.ics.uci.edu/ml/machine-learning-databases/adult/adult.names)

## 2. 변수 설명

| 변수명 | 자료형 | 역할 | 설명 |
|---|---|---|---|
| `age` | 수치형 | 입력 변수 | 나이 |
| `workclass` | 범주형 | 입력 변수 | 근로 형태 또는 고용 부문. 예: 민간기업, 자영업, 연방정부, 지방정부 |
| `fnlwgt` | 수치형 | 입력 변수 | 표본의 최종 가중치. 해당 행과 비슷한 특성을 가진 사람이 모집단에 어느 정도 있는지를 반영한 값 |
| `education` | 범주형 | 입력 변수 | 최종 학력. 예: 고등학교 졸업, 학사, 석사, 박사 |
| `education-num` | 수치형 | 입력 변수 | 학력 수준을 숫자로 나타낸 값. `education`과 같은 정보를 다른 형태로 표현한 변수 |
| `marital-status` | 범주형 | 입력 변수 | 혼인 상태. 예: 기혼, 미혼, 이혼, 별거, 사별 |
| `occupation` | 범주형 | 입력 변수 | 직업군. 예: 사무직, 기술지원, 판매직, 관리직 |
| `relationship` | 범주형 | 입력 변수 | 가구 안에서의 관계. 예: 남편, 아내, 자녀, 비가족 구성원 |
| `race` | 범주형 | 입력 변수 | 인종 범주 |
| `sex` | 범주형 | 입력 변수 | 성별 |
| `capital-gain` | 수치형 | 입력 변수 | 자본 이득 금액 |
| `capital-loss` | 수치형 | 입력 변수 | 자본 손실 금액 |
| `hours-per-week` | 수치형 | 입력 변수 | 주당 근무 시간 |
| `native-country` | 범주형 | 입력 변수 | 출신 국가 |
| `income` | 범주형 | 목표 변수 | 연간 소득 구간. `<=50K`는 50,000달러 이하, `>50K`는 50,000달러 초과 |

## 3. 결측치 처리

원본 파일은 결측치를 빈칸이 아니라 `?`로 표시한다. 데이터를 읽을 때
`?`를 결측값으로 변환했다.

결측값은 다음 세 변수에 존재한다.

| 변수명 | 결측값 수 |
|---|---:|
| `workclass` | 1,836개 |
| `occupation` | 1,843개 |
| `native-country` | 583개 |

나머지 변수의 결측값은 0개다. 결측값이 하나라도 있는 행을 제거한 결과 데이터는
32,561행에서 30,162행으로 줄었다. 여기에 완전히 같은 중복 행까지 제거한
최종 데이터 30,139행을 이후 분석에 사용한다.
한 행에 둘 이상의 결측값이 함께 있을 수 있으므로 각 열의 결측값 수를 더한 값과
실제로 제거된 행 수는 같지 않다.
데이터를 불러올 때 쉼표 다음의 공백도 함께 제거했다.

## 4. 분석 목표

- 연령, 학력, 직업, 주당 근무 시간 등의 특성에 따라 소득 구간에 어떤 차이가 있는지 확인한다.
- 두 소득 구간의 크기와 주요 특성을 비교한다.
- 성별, 근로 형태, 학력별로 연간 소득 50,000달러 초과 비율을 비교한다.
- 각 변수와 연간 소득 50,000달러 초과 여부의 관계를 표와 그래프로 비교한다.
- 결측값을 처리하고 범주형 변수를 모델에 사용할 수 있는 형태로 변환한다.
- 개인의 특성을 이용해 연간 소득이 50,000달러를 초과하는지 예측하는 분류 모델을 만든다.
- 정확도 등의 평가 지표를 이용해 모델의 예측 성능을 확인한다.
